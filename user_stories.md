# Patient Appointment Portal - User Stories

## Admin User Stories
* As an Admin, I need to add, update, or remove doctor profiles so that the portal's directory remains accurate.
* As an Admin, I need to manage system access levels so that patient records and doctor schedules remain secure.
* As an Admin, I need to view system-wide appointment statistics so that I can monitor the hospital's daily operations.

## Patient User Stories
* As a Patient, I need to view available appointment slots for specific doctors so that I can schedule a visit at a convenient time.
* As a Patient, I need to cancel or reschedule an existing appointment so that the slot frees up for someone else if my plans change.
* As a Patient, I need to view my upcoming appointments and past medical history so that I can keep track of my healthcare.

## Doctor User Stories
* As a Doctor, I need to view my daily and weekly appointment schedule so that I can prepare for my upcoming patients.
* As a Doctor, I need to update my working hours and availability so that patients cannot book slots when I am off-duty.
* As a Doctor, I need to access my patient's previous consultation notes so that I can provide better continuous care.
